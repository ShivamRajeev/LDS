# LDS Website

A modern premium architectural lighting website built with React, Vite, Tailwind CSS, React Router, and Framer Motion.

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Framer Motion

## Features

- Premium dark luxury visual style with warm golden accents
- Fully responsive layouts for desktop, tablet, and mobile
- 10 core pages with reusable component architecture
- Product and project listing with detail pages
- Product filter sidebar and interactive product finder
- Smooth animation for hero, cards, buttons, and page transitions
- Semantic structure and runtime SEO title/description updates
- Local mock data for products and projects

## Project Structure

- src/components
- src/pages
- src/data
- src/assets
- src/routes
- src/App.jsx
- src/main.jsx
- tailwind.config.js
